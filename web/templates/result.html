Term Meaning
b1, b2, b4 Bit layers — refer to the specific bits (usually least significant bits) used to embed hidden data.
r / g / b / rgba Color channels — Red, Green, Blue, and Alpha (transparency) used in image encoding. Data may be hidden in specific channels.

meta Metadata — refers to image metadata fields (like EXIF tags) where messages can be hidden.
text Human-readable output — indicates a hidden message was found as plaintext.
file Binary file signature — indicates a recognizable file format (e.g., ZIP, PNG) embedded within the image.
